# Sankirtan - Bhajan Se Bhagwan Tak

The devotional music platform for singers, artists & devotees.

**Founder:** Surendra Jain  
**Website:** https://sankirtan.app  
**Firebase Project:** sankirtan-app-ebc18

## Session 1 - Foundation

This is the foundation of Sankirtan SaaS with:

- ✅ Google Sign-In authentication
- ✅ Phone OTP authentication  
- ✅ User profile creation
- ✅ Beautiful welcome screen
- ✅ Personalized dashboard
- ✅ Firebase Firestore integration
